# Desafio Full Cycle - Clean Architecture: Use Cases para a Entidade Product

Implementação de modelagem tática com DDD, padrões de projeto e testes unitários utilizando Typescript.

## Tabela de Conteúdos

- [Requisitos](#requisitos)
- [Instalação](#instalação)
- [Executar testes](#executar-testes)


## Requisitos

- Node.js
- TypeScript
- Jest

## Instalação

1. **Clone o repositório**:

   ```bash
      git clone https://github.com/carlosrsantos/fc-clean-architecture.git

      cd <app-folder>
   ```

2. **Instalar dependências**:
  
   ```bash
      npm install
   ```

## Executar testes

1. **Executar todos os testes**:

   ```bash
      npm run test
   ```

2. **Executar um arquivo de testes específico**:

   ```bash
      npm run test:file -- <nome_do_arquivo_de_teste.spec.ts>
   ```

3. **Executar um ou vários testes a partir do nome do teste**:

   ```bash
      npm run test:file -- -t "nome do teste"
   ```
